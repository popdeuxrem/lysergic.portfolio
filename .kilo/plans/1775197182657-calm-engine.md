## Implementation Plan

### Phase 1: Physics Engine Integration (CRITICAL)

**Task 1.1: Create PhysicsWorld wrapper**
- Create new `src/components/PhysicsWorld.jsx`
- Wrap scene with `<Physics gravity={[0, -9.81, 0]}>`
- Add ground plane as fixed RigidBody

**Task 1.2: Convert Vehicle to Rapier RigidBody**
- Replace JS velocity movement with Rapier `RigidBody` component
- Use `useRapier()` hook for vehicle controller
- Maintain acceleration/maxSpeed/friction tuning for Bruno feel

**Task 1.3: Add Environment Colliders**
- Wrap environment model with `<RigidBody type="fixed" colliders="trimesh">`
- Fallback: procedural ground has RigidBody already

**Task 1.4: Update vite.config.js**
- Ensure manualChunks includes rapier for proper bundling

### Phase 2: Scene Replication

**Task 2.1: Camera Follow System**
- Current CameraRig.jsx is adequate
- Refine lerp values for smoother follow feel
- Add camera shake on collision (optional)

**Task 2.2: Trigger Zones**
- Current TriggerZones.jsx works via distance check
- Add collision-based triggers as backup

### Phase 3: UI Enhancement (Terminal)

### Phase 4: Visual Enhancement
